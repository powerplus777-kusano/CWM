### 3X3マトリクステーブル
- 中央セルはテーマタイトル
- 周囲8セルは各項目（クリックでポップアップ詳細表示）
- グラデーション背景色で視認性向上

### 3つのアクションボタン
1. **📄 マトリクス表を印刷** - マトリクス表のみ印刷
2. **📑 一括印刷** - マトリクス表+全詳細ページを印刷
3. **💾 一括HTMLダウンロード** - マトリクス表+全詳細ページをHTMLファイルとしてダウンロード

### 印刷最適化（重要！）
#### マトリクス表はA4サイズ1ページに収まるよう調整

**導入テキストなし:**
- h1: 24px
- padding: 10px
- font-size: 11px
- line-height: 1.5
- min-height: 100px

**導入テキストあり（よりコンパクト）:**
- h1: 20px
- intro: 11px / 10px（内容量により調整）
- padding: 8px
- font-size: 10px
- line-height: 1.4
- min-height: 100px

### 印刷CSSのポイント
```css
@media print {
    /* マトリクス表のみ印刷モード */
    .matrix-only-mode #detailPages {
        display: none !important;
    }

    /* 一括印刷モード */
    .print-all-mode #detailPages {
        display: block !important;
    }

    .print-all-mode .detail-page {
        page-break-before: always;
        display: block !important;
    }

    .print-all-mode #hero {
        page-break-after: always;
    }
}

/* ブラウザ表示でのページ区切り（印刷時は適用しない） */
@media screen {
    .print-all-mode .matrix-table {
        margin-bottom: 100vh;
    }

    .print-all-mode .detail-page {
        min-height: 100vh;
        padding-top: 50px;
        display: block !important;
    }
}
```

## 🚨 重要な問題解決履歴

### 問題1: HTMLダウンロード時に詳細ページが含まれない
**解決策:** downloadHTML()関数で、詳細ページを一時的に表示してから全体のHTMLを取得

### 問題2: ダウンロードしたHTMLに改ページがない
**解決策:** @media screenでブラウザ表示用のスタイルを分離し、印刷時には@media printのみ適用

### 問題3: マトリクス表がA4に収まらない
**解決策:** フォントサイズ、パディング、行間を調整。導入テキストありの場合はさらに小さく設定
