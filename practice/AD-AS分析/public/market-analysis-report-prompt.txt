# 日本企業の市場動向分析レポート作成プロンプト

## プロジェクト概要

日本企業の市場動向を包括的に分析するワンページ完結型のレポートサイトを作成してください。
最新の統計データ（2024-2025年）に基づき、企業数、売上高、税収、人口動態、労働力の関係性を詳細に解説します。

## 対象ファイル

- 保存先: public/market-analysis-report.html
- エンコーディング: UTF-8

## レポート構成（セクション）

### 1. Hero Section
- タイトル: 「日本企業の市場動向分析レポート 2024-2025年版」
- サブタイトル: 企業数・売上・税収の関係性から読み解く日本経済の現状と未来
- 背景: グラデーション（ブルー系）
- CTAボタン: レポート閲覧へのスクロール

### 2. エグゼクティブサマリー
- レポートの目的と概要
- キーメトリクス（3つの数値ハイライト）
  - 日本の総企業数
  - 中小企業の占める割合
  - インボイス登録件数

### 3. セクション1: 企業数の実態
- タイトル: 「1. 日本企業の規模別構成」
- 内容:
  - 企業規模別の構成（大企業 vs 中小企業）
  - 中小企業の詳細内訳（小規模・中規模）
  - 業種別の企業数分布
  - データの視覚化（数値ボックス、SVGグラフ等）

### 4. セクション2: 売上高の分析
- タイトル: 「2. 企業規模別の売上構造」
- 内容:
  - 企業規模別の売上高と構成比
  - 1社あたりの平均売上高
  - 売上集中度の分析
  - 中小企業の売上における課題

### 5. セクション3: 個人事業主の実態
- タイトル: 「3. 個人事業主と小規模事業者」
- 内容:
  - 個人事業主の数と推移
  - 課税所得の分布
  - フリーランス・ギグワーカーの増加
  - 小規模事業者の課題

### 6. セクション4-1: 人口動態
- タイトル: 「4-1. 人口動態と経済への影響」
- 内容:
  - 総人口の推移（現在～2070年予測）
  - 年齢構成の変化（生産年齢人口、高齢者人口）
  - 人口減少が経済に与える影響
  - 地域別の人口動態

### 7. セクション4-2: 労働力（生産人口）の推移
- タイトル: 「4-2. 労働力(生産人口)の推移と市場への影響」
- 内容:
  - 生産年齢人口の減少トレンド
    - ピーク時（1995年）: 8,716万人
    - 現在（2024年）: 7,395万人
    - 予測（2070年）: 4,535万人
  - 雇用形態の変化
    - 正規雇用 vs 非正規雇用の推移と割合
    - 非正規雇用の増加要因
  - 団塊世代の大量退職の影響
    - 退職時期: 2012-2014年
    - 人数: 約800万人
    - 労働市場への影響
  - 男女別の労働力参加率
    - 女性就業率の向上（50.1% → 68.2%）
    - M字カーブの解消状況
  - 各データをSVGグラフや数値ボックスで視覚化

### 8. セクション4-3: 労働力不足への対応策
- タイトル: 「4-3. 労働力不足への対応策」
- 内容:
  - 超高齢社会への対応
    - 高齢化率: 29.1%（2024年）
    - 70歳就業機会確保の努力義務化
    - 健康寿命の延伸（男性72.68歳、女性75.38歳）
    - 生涯現役社会の実現
  - 後継者不足問題と事業承継支援
    - 中小企業経営者の平均年齢: 62.5歳
    - 後継者未定の企業: 65%
    - M&A・事業承継支援策
    - 実例: EBO（従業員承継）の活用事例
  - 女性の活躍推進
    - 女性就業者数: 3,032万人（全体の45.2%）
    - 管理職に占める女性割合: 13.2%（先進国最低水準）
    - 育児支援制度の拡充
    - 柔軟な働き方の推進
  - DX（デジタルトランスフォーメーション）の推進
    - デジタル競争力ランキング: 世界27位
    - DX投資額: 7.8兆円
    - 生産性向上効果: 20-30%
    - 中小企業のDX課題と支援策
  - 各対応策を4つの柱として色分けして表示

### 9. セクション5: インボイス制度の影響
- タイトル: 「5. インボイス制度の影響」
- 内容:
  - インボイス制度の概要
  - 登録事業者数の推移
  - 免税事業者への影響
  - 今後の課題と対応策

### 10. セクション6: 税収と使途
- タイトル: 「6. 税収構造と財政」
- 内容:
  - 国税・地方税の構成
  - 法人税収の推移
  - 税収の使途（社会保障、公共事業等）
  - 財政健全化の課題

### 11. セクション7: 税金の使い道
- タイトル: 「7. 税金の使い道と社会還元」
- 内容:
  - 予算配分の詳細
  - 社会保障費の内訳
  - インフラ投資
  - 教育・科学技術への投資

### 12. セクション8: 今後の展望
- タイトル: 「8. 今後の市場展望と提言」
- 内容:
  - 人口減少社会における企業の戦略
  - DX・イノベーションの必要性
  - 持続可能な成長に向けて
  - 具体的な提言（3-5項目）

### 13. Footer
- レポート情報（作成日、データソース）
- 免責事項
- 連絡先情報（オプション）

## デザイン要件

### カラーパレット
- Primary: #1e3a8a（濃紺）
- Secondary: #3b82f6（ブルー）
- Accent: #60a5fa（ライトブルー）
- Background: グレー系（#f1f5f9）
- テキスト: #0f172a（ダークグレー）

### レイアウト
- ワンページ完結型
- セクションごとに明確な区切り
- 交互に背景色を変更（白 ↔ グレー）
- 適度なホワイトスペース

### データ表示
- 数値は大きく目立つように（text-4xl以上）
- 重要な統計は色付きボックスで強調
- グラフはSVGで作成（シンプルな棒グラフ、折れ線グラフ）
- アイコンはLucide Icons使用

## 技術要件

### HTML構造
- セマンティックHTML5
- 各セクションにID属性（#hero, #companies, #sales, #individual, #population, #labor-force, #labor-solutions, #invoice, #tax-revenue, #tax-usage, #future）
- メタタグ完備（title, description, keywords, OGP）

### CSS/フレームワーク
- Tailwind CSS v4.1以上（CDN使用）
- Animate.css v4.1.1（適度なアニメーション）
- AOS v2.3（スクロールアニメーション）
- Google Fonts - Noto Sans JP

### ナビゲーション
- スティッキーヘッダー
- 初期状態: 透明背景（白文字、テキストシャドウ）
- スクロール後: 半透明白背景（backdrop-filter: blur(12px)）、濃色文字
- レスポンシブ対応（タブレット以下でハンバーガーメニュー）
- 主要セクションへのリンク

### レスポンシブ対応
- モバイルファースト設計
- ブレークポイント: sm (640px), md (768px), lg (1024px), xl (1280px)
- 画像に loading="lazy" decoding="async"

### 印刷CSS（必須）
```css
@media print {
    /* 1. ナビ・フッター非表示 */
    nav, .no-print, .site-header, .site-footer {
        display: none !important;
    }

    /* 2. アニメーション無効化 */
    [data-aos], .animate__animated {
        opacity: 1 !important;
        transform: none !important;
        animation: none !important;
    }

    /* 3. セクションごとに改ページ */
    #hero {
        page-break-after: always;
    }

    #companies, #sales, #individual, #population, #labor-force, #labor-solutions, #invoice, #tax-revenue, #tax-usage, #future {
        page-break-before: always;
    }
}
```

### 避けるべき印刷CSS
- ❌ page-break-inside: avoid（大きなセクションに適用しない）
- ❌ @page { size: A4 portrait; }（用紙の向きを固定しない）
- ❌ 背景色・フォントサイズの強制変更
